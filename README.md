# policy-visualizer-jp

日本の政策テーマを、解説・グラフ・シミュレーターで可視化するプロジェクトです。

各テーマは個別ページとして管理し、トップページから一覧で辿れる構成にしています。  
外部で生成した静的HTMLを取り込むケースと、テンプレートから新規作成するケースの両方に対応しています。

## 現在の公開テーマ

- 太陽光・蓄電池 vs 原子力コスト比較
- 月面・宇宙基地建設コスト試算
- 会社員から個人事業主へ 社会保障リスク分析

## ディレクトリ構成

- `index.html`: トップページ
- `policies/`: 個別政策ページ
- `assets/`: 共通スタイル
- `templates/`: 新規ページ作成用テンプレート
- `docs/`: 運用メモ・ガイド

## 方針

- 各政策ページは `policies/<slug>/index.html` に配置
- トップページから辿れることと、個別ページへの直リンクの両立を重視
- 外部生成の静的コードもそのまま取り込み可能
