# 今後の政策テーマ案 (Backlog)

このドキュメントでは、今後「政策解説シミュレータ」に追加する可能性のあるテーマやアイデアを管理します。

## 候補一覧

### 1. 少子高齢化・人口動態シミュレーター
- **タイトル案**: 「20XX年の日本を救え：人口構造と現役世代の負担率」
- **内容**: 合計特殊出生率や移民受け入れ数、定年延長のパラメータを操作して、将来の現役世代の負担（肩車型社会）を可視化。
- **可視化**: 人口ピラミッドアニメーション、現役世代の可処分所得推移。

### 2. 空き家問題・地方自治体財政
- **タイトル案**: 「あなたの街が消える日：空き家率と公的インフラ維持コスト」
- **内容**: 空き家率の増加が水道・道路などのインフラ維持費や自治体財政に与える影響をシミュレート。
- **可視化**: 地域マップ上のインフラ維持限界点。

### 3. 子育て支援・教育無償化の投資対効果
- **タイトル案**: 「10兆円の使い道：教育への投資 vs 現金給付の将来リターン」
- **内容**: 児童手当、大学無償化、保育士待遇改善など、異なる政策が30年後のGDPや税収にどう寄与するかを比較。
- **可視化**: 人的資本への投資と経済成長の相関。

### 4. カーボンニュートラル・生活コスト
- **タイトル案**: 「脱炭素社会の家計簿：EV・断熱リフォームの損益分岐点」
- **内容**: EV切り替えや住宅断熱改修の初期投資が、将来の光熱費削減で何年で回収できるかをシミュレート。
- **可視化**: 損益分岐点グラフ。

### 5. 防衛費・増税・国債のトレードオフ
- **タイトル案**: 「国家予算のパズル：防衛費増額と他予算（医療・介護）の削減シミュレーター」
- **内容**: 防衛費増額の財源（増税・国債・予算削減）の組み合わせによる経済・社会保障への影響を検討。
- **可視化**: 財政健全性と国防能力のバランス。
